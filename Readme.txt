This project is a prototype for college management system developed by Manjyot Singh.

This project illustrates different functionalities for 2 modules:
1.)Admin 
2.)Student

When a new student is enrolled in college, a new account is added by admin in the databse.

This project also contains a database file which is needed in order to execute php scripts.

To execute the project you'll need Xampp apache and mysql server.
 
 The next step is to launch the project by navigating to index.php.
